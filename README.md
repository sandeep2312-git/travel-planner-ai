# Travel Planner AI (Streamlit)

A Streamlit travel itinerary generator that works:
- without any API key (rule-based itinerary)
- with OpenAI (optional AI itinerary)

## Deploy (Streamlit Community Cloud)
- Main file: `app.py`
- Requirements: `requirements.txt`

## Optional AI Mode
Add Streamlit Secrets:
- OPENAI_API_KEY = "your_key"
- (optional) OPENAI_MODEL = "gpt-4o-mini"
